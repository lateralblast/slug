SLUG
====

Set Laptop Up Gracefully

Introduction
------------

A script to help automate the setup of a Mac

At the moment it is a simple shell script

Usage
-----

```
Usage ./setup.sh -abszrpdgVl

-h: print help
-V: print version
-a: do everything
-b: do brew packages
-s: do other packages
-z: do shell setup
-r: do ruby setup
-p: do python setup
-d: do defaults
-g: do go setup
-l: list packages
```
